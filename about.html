About this Dollmaker

This is a dollmaker for creating sprites for use with Renpy and other visual novel engines, or just for fun. It is currently a very early alpha build and super unreliable! Saves made now will definitely break in later releases.

Please let me know if you hit any bugs or if there's any extra features you'd like, though I make no guarantees.

Possible Future Features

  • Wheelchair option

Known Bugs

  • Occasional flicker while image is updating
  • Sometimes fills the image with colour when stubble is randomly generated. If you reset the facial hair this fixes itself.
  • Colours sometimes take a second or two to 'catch up' to the right value.
  • Import and right-click-save don't work on all browsers. Chrome and Firefox seem to be ok.
  • Colour picker is sometimes slightly off.
